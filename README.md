**Requirements:** 
Java, Spring, and Maven. Jdk 17, Springboot 3.3.0, Maven 3.9.7

**Dependency:** 
Spring Web, Lombok, Spring Data JPA, H2 Database

**How to Run Project**
- Download Source
- open in eclips/visual studio code/intelliJ or IDE
- Check Requirements
- mvn clean install
- mvn spring-boot:run
- check api in localhost:8080/{endpoint}
